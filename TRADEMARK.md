> 🏷️ The name and logo are part of the project identity. They are protected separately from the code license.

"NextBitDeskify" and the NextBitDeskify logo are trademarks of Aquila Wilfred and the
NextBitDeskify project. The code license (MIT) covers the source code — it does
not grant any rights to the project name, logo, or branding. Open source
licenses grant copyright permissions, not trademark permissions.

## In Plain English

- Use your own brand name for forks or derived products.
- Do not present a fork as the official NextBitDeskify project.
- Feel free to say a project is "based on NextBitDeskify" when it is clearly distinct.

We want users to trust that something called "NextBitDeskify" really is this
project. So if you publish a fork or a derived product, please:

- Use your own name and icon — not "NextBitDeskify" or the NextBitDeskify logo.
- Don't imply your product is endorsed by, affiliated with, or officially
  connected to NextBitDeskify.
- Don't use the NextBitDeskify name to market a paid or competing product.
- If your fork is a direct derivative of this codebase, a note such as
  "based on NextBitDeskify" is welcome, but the fork must still use its own
  distinct name and branding.

Permission requests: open an issue at
<https://github.com/AquilaWilfred/NextBitDeskify>.

## Why a strong identity matters

- Trust and recognition: a clear project identity helps people remember and recommend NextBitDeskify more easily. Evidence: the project uses a consistent banner, naming, and documentation style across the repository.
- Better community growth: a well-defined brand makes the project easier to discuss, share, and build around. Evidence: the repo includes dedicated docs for contribution, licensing, and policy.
- More confidence for adopters: clear boundaries around the name and logo help users understand what is official and what is a fork or derivative work. Evidence: the trademark policy explains those distinctions directly.

Supporting the project with good usage, attribution, and respectful sharing helps the brand grow in a healthy way.
